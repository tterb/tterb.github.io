Atom File Icons



Description:

I’ve often found that looking through the files of a large project in a file manager can be a tedious process, worsened by the added difficult of discerning different file-types. In response, I’ve created this collection of icons to be tastefully unique, while also providing improved visual grepping that allows for much quicker navigation of project files.

Note: If you’d like to request an icon for a file-type that isn’t already included in the collection, make sure to submit your suggestions here. Additionally, while I do my best to fulfill any submitted icon requests, feel free to create/contribute your own using the provided Illustrator file.
